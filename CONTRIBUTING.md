# Contributing

Thank you for your interest in Pristan!

This is a **read-only informational repository**. The Pristan platform is developed privately and we do **not** accept external contributions, pull requests, or issues on this repository.

## How to Get Involved

If you're interested in Pristan for your organization:

- **Website:** [pristan.chat](https://www.pristan.chat)
- **Email:** [hello@pristan.chat](mailto:hello@pristan.chat)
- **Request a demo:** [pristan.chat/enterprise](https://www.pristan.chat/enterprise)

## Bug Reports & Feature Requests

If you're a Pristan customer and need to report a bug or request a feature, please use the support channels provided with your deployment.

## Security Vulnerabilities

If you've discovered a security vulnerability, please report it responsibly via email to [security@pristan.chat](mailto:security@pristan.chat). Do not open a public issue.
