# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability related to Pristan, please report it responsibly.

**Email:** [security@pristan.chat](mailto:security@pristan.chat)

Please include:

- Description of the vulnerability
- Steps to reproduce (if applicable)
- Potential impact

We will acknowledge receipt within 48 hours and provide a timeline for resolution.

**Do not** open public issues for security vulnerabilities.

## Scope

This security policy covers the Pristan platform and its public-facing properties:

- pristan.chat (website)
- app-01.pristan.chat (application)
- This repository

## Security Features

For information about Pristan's security architecture, visit [pristan.chat/security](https://www.pristan.chat/security).
